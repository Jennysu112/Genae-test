# GENAE · Jelly — Static Site

This repository hosts a simple static landing site for **GENAE · Jelly** with bilingual pages:

- `index.html` — English (default)
- `zh.html` — Traditional Chinese

## Local preview
Just open the HTML files in your browser.

## Deploy to GitHub Pages
1. Create a public repo (e.g., `genae-jelly`).
2. Upload the files in this folder to the repo root.
3. In **Settings → Pages**, set **Source** to `Deploy from a branch`, then pick the default branch root.
4. Your site will be available at `https://<your-username>.github.io/genae-jelly/`.
